Site PETComp

Esse projeto foi desenvolvido para seletivo PETComp UFMA.
É um site para um evento de computação, incluindo cronograma e 
inscrições para o evento.

O trabalho foi desenvolvido utilizando HTML, CSS e JavaScript.
Pode ser executado em qualquer navegador.

O site está hospedado na platfaforma Vercel e 
pode ser acessado através do link: https://site-pet-comp.vercel.app/ .

Esse projeto foi desenvolvido apenas para fins educacionais.
